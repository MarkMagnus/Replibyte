# Databases

Replibyte supports [PostgreSQL](#postgresql), [MySQL](#mysql--mariadb)

## PostgreSQL

:::caution requirements

You need `pg_dump` binary locally installed for running `replibyte backup --remote` and `replibyte restore --remote`

:::

To use PostgreSQL it's as simple as using prefixed connection URI with `postgres://`

```yaml
source:
  connection_uri: postgres://<user>:<password>@<host>:<port>/<database> # you can use $DATABASE_URL
#...
destination:
  connection_uri: postgres://<user>:<password>@<host>:<port>/<database> # you can use $DATABASE_URL
```
